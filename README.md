# wbar

A minimalist status bar for Windows, designed to pair with [GlazeWM](https://github.com/glzr-io/glazewm).

Inspired by [zebar](https://github.com/glzr-io/zebar) but without the webview / React build pipeline — just a single TOML config file and a small set of built-in widgets.

## Status

Early development. Not yet usable.
